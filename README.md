#cricket_roulette_spin

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To run in dev mode:

```bash
bun run --watch index.ts |
bun --watch index.ts


```

This project was created using `bun init` in bun v1.1.30. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
